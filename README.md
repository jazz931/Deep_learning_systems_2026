# DeepFace — Анализ демографических и эмоциональных характеристик лица

Данный проект предназначен для автоматического определения возраста, пола и доминирующей эмоции человека по одной фотографии с использованием библиотеки **DeepFace**.

Проект адаптирован для работы на центральном процессоре (CPU) и полностью контейнеризирован с использованием Docker для обеспечения воспроизводимости результатов на любой платформе.

---

## 1. Ресурсы

Для корректного запуска проекта необходимо:

* **Операционная система**: Windows 10/11 с поддержкой WSL2 **или** Linux/macOS с установленным Docker.
* **Свободное место**: не менее **10 ГБ** на диске (для образа Docker и кэша моделей ~1 ГБ).
* **Программное обеспечение**: 
    * Docker Desktop (версия 4.67+) — [скачать](https://www.docker.com/products/docker-desktop/)
    * Интернет-соединение (для первой загрузки весов моделей)

## 2. Инструкция по запуску

### Шаг 1: Склонировать репозиторий
  ```console
  git clone https://github.com/jazz931/Deep_learning_systems_2026
  ```

### Шаг 2: Перейти в папку проекта
  ```console
  cd deepface
  ```

### Шаг 3: Запустить Docker Desktop
* Открой приложение Docker Desktop из меню Пуск
* Дождись зелёной надписи "Engine running" в левом нижнем углу

### Шаг 4: Собрать Docker-образ
  ```console
  docker build -t deepface_zachet .
  ```
* Первая сборка займёт 5–10 минут (скачиваются базовые образы и зависимости). Повторные сборки выполняются значительно быстрее благодаря кэшированию слоёв.

### Шаг 5: Запустить тест с сохранением результатов
  ```console
  docker run -v %CD%/output:/app/output -v %CD%/.deepface_weights:/root/.deepface/weights deepface_zachet
  ```
* Для Linux/macOS используй $(pwd) вместо %CD%:
  ```console
  docker run -v $(pwd)/output:/app/output -v $(pwd)/.deepface_weights:/root/.deepface/weights deepface_zachet
  ```

## 5. Возможные проблемы и решения
* Could not find cuda drivers - это нормально, проект работает на CPU, GPU не требуется
* Долгий первый запуск (~3 мин) - скачиваются модели (~1 ГБ), второй запуск будет за сильно меньшее время
* ImgNotFound или File not found - положи фото в test_input/my_face.jpg
* docker: command not found - установи Docker Desktop и добавь его в PATH
* Ошибка при docker build - проверь, что Docker Desktop запущен и «Engine running»

## Примечание:
* Изображение test_input/my_face.jpg используется только для демонстрации работы автотеста. Для проверки на других данных замените файл и обновите эталонные значения в RunCheckerTest.py
